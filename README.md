# 💰 Expense Tracker App

![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=nextdotjs&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

A modern, responsive, and aesthetically pleasing Expense Custom Tracker Web Application built with **Next.js 15**, **React 19**, and **Tailwind CSS**. Easily track your financial health, monitor daily expenses, visualize data with dynamic charts, and work towards your savings goals!

---

## ✨ Features

- **📊 Comprehensive Dashboard:** Get a bird's-eye view of your finances with total balance, expenses, and active income.
- **🎯 Savings Goals:** Set targets (e.g., 50,000 INR) and visually track your progress towards them.
- **📈 Interactive Data Visualization:** Clean and responsive charts (built with Recharts) to analyze spending trends over time.
- **✨ Fluid Animations:** Delightful UI/UX micro-interactions powered by Framer Motion.
- **📱 Fully Responsive:** Carefully crafted with Tailwind CSS to look pixel-perfect on mobile, tablet, and desktop devices.
- **🌓 Modern UI:** Dark/Light aesthetics tailored for a premium user experience.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Charts:** [Recharts](https://recharts.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Utilities:** `date-fns` for date manipulation, `clsx` & `tailwind-merge` for style merging.

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

You need to have **Node.js** (v18 or higher) and **npm/yarn/pnpm** installed.

### Installation

1. **Clone the repository** (if applicable):

   ```bash
   git clone https://github.com/Rahul0070050/expense-tracker-app.git
   cd expense-tracker
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open the app:**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
